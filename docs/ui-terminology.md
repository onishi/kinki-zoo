# UI 用語集・表記ルール

UI に使う言葉の基準をまとめます。完璧な統一より、迷ったときの判断を減らすことを目的にします。

---

## 施設・動物園の呼び方

| 場面 | 使う言葉 | 使わない言葉 |
|------|----------|-------------|
| 画面タイトル・ナビゲーション | **動物園一覧** | 施設一覧 |
| サマリー文・件数表示 | **動物園・施設** | 動物園・動物施設 |
| 動物が見られる場所を指すとき | **施設** | 動物園（動物園以外の施設も含むため） |
| テーブルの列見出し | **施設名**、**施設数** | 動物園名、動物園数 |
| パンくずリスト | **動物園一覧** | 施設一覧 |
| 戻るリンク | **動物園一覧へ戻る** | 施設一覧へ戻る |

**判断基準：**
- トップページ（`/`）を指すリンクやタイトルは **動物園一覧**
- 一般的に「場所」として言及するときは **施設**（動物王国・ファームパーク等も含むため）
- サマリー文の `N 件` 前後には **動物園・施設** を使い「動物園・動物施設」は使わない

---

## 動物数の表記

| 場面 | 使う言葉 | 使わない言葉 |
|------|----------|-------------|
| テーブル列見出し | **動物種数** | 動物数、登録表示名 |
| 動物ページのクイック情報 | **動物種数** | 総動物数 |
| 探し方カードのメタ情報 | **動物 N 種** | 登録表示名 N 件 |
| スポットライトカード | **N 種** | N 件 |

**判断基準：**
- 「種」は動物学的分類でなく「種類」の意。表記は **N 種** で統一
- 「登録表示名」は内部 DB 用語。利用者向けの画面では使わない
- 施設ごとのスクレイピング件数も外部向けは **N 種** と表示する

---

## 表示名・分類名

| 用語 | 定義 |
|------|------|
| **表示名**（display_name） | 各施設が公式に使っている動物の呼び名。「ニホンザル」「おかあさんゾウ」など |
| **代表名**（canonical_name） | 分類マスタでその種に対応する標準的な和名 |
| **分類** | 類・目・科・属・種の階層。利用者向けには「分類から探す」と表記 |

利用者向けの説明文では「表示名」「代表名」は使わず、単に「動物名」と書く。

---

## 利用者向けと管理者向けの言葉

### 利用者向け（一般公開ページ）

| 避ける言葉 | 使う言葉 |
|-----------|---------|
| 登録表示名 | 動物名 / 動物 N 種 |
| スクレイピング | （説明しない） |
| D1 / キャッシュ | （説明しない） |
| animal_id | （説明しない） |
| 分類マスタ | 分類 |

### 管理者向け（`/admin/*`、`/animal-images`）

管理者向けページでは技術的な用語（表示名、分類マスタ、スクレイピング等）をそのまま使ってよい。

---

## よく使う文言

### 空状態メッセージ

| 状況 | 文言 |
|------|------|
| 動物検索でヒットなし | 検索条件に該当する施設が見つかりませんでした。 |
| 施設一覧で表示なし | 該当する施設が見つかりませんでした。 |
| 動物一覧がまだない | 動物データがまだありません。動物園一覧から気になる施設を開き、掲載動物をご確認ください。 |
| 施設の動物が未取得 | 動物一覧をまだ取得できていません。最新の情報は公式サイトでご確認ください。 |

### ボタン・リンク

| 用途 | 文言 |
|------|------|
| 検索実行 | 検索 |
| 検索クリア | クリア / 検索をクリア |
| 一覧ページへ戻る | 動物園一覧へ戻る |
| 外部リンク（公式サイト） | 公式サイトを見る |

### フッター

```
データは各施設の公式情報をもとに作成。最新情報は各施設の公式サイトでご確認ください。
```
